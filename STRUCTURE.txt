QUICK START
===========

Backend Setup:
-------------
cd backend
npm install
cp .env.example .env
npm run db:generate
npm run dev

Frontend Setup:
--------------
cd frontend
npm install
npm run dev


DEPLOYMENT
==========

Backend runs on: http://localhost:3000
Frontend runs on: http://localhost:5173

Deploy Backend:
- Railway: railway up
- Vercel: vercel deploy
- Docker: docker-compose up

Deploy Frontend:
- Vercel: vercel deploy
- Build: npm run build
